# **App Name**: PortfolioPro Admin

## Core Features:

- Dashboard Home: Dashboard overview with project and skill counts.
- Project Management: CRUD operations for projects using react-data-table-component.
- Skill Management: CRUD operations for skills using react-data-table-component.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional look.
- Secondary color: Teal (#009688) to complement the primary color.
- Accent: Amber (#FFC107) for highlights and action items.
- Clean and modern typography for readability.
- Use Material Design icons for a consistent UI.
- Card-based layout for displaying project and skill summaries.
- Responsive layout to adapt to different screen sizes.
- Subtle transitions and animations for a smooth user experience.
